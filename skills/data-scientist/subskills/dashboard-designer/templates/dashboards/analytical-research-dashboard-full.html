Risks, Caveats & Assumptions

High — Metric gap

Segment G (underrepresented) shows recall 10–15 pp below cohort average. Disparity is statistically significant (p < 0.01 via bootstrap). Recommend segment-level audit before full deployment to this segment.

High — Small-sample segment

Segment F (N = 4,100) is below the 5,000 minimum threshold. Recall is unstable (±0.09 across windows). Retrain or merge with parent segment before relying on this split.

Medium — Feature drift

Three input features show distribution shift between training and validation windows (JS divergence > 0.15): feature_a, feature_b, feature_c. Monitored; no degradation in aggregate metrics yet.

Low — Imputation

0.7% of feature rows are imputed via median (attribute: feature_x). Sensitivity analysis shows ±0.003 AUC impact under mean, median, and mode substitution. Not material at current volume.

Low — Temporal leakage check

All features verified to precede label by ≥24 hours. Backfill window of 48 hours used for batch features. No label-to-feature leakage detected in train/val split audit (sequential split, no random shuffle).

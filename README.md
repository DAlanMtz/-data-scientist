# Data Scientist Skill

A senior, tool-flexible OpenCode skill for data mining, exploratory analysis, model building, validation, interpretation, business decision-making, responsible AI, and production-readiness.

The skill supports practical data science work across Python, R, SQL, Excel/Sheets, notebooks, scripts, databases, dashboards, APIs, and production pipelines. It is not tied to a course, language, or single modeling tool.

## What It Covers

- Exploratory data analysis
- Data quality checks
- Feature engineering
- Model selection
- Regression
- Classification
- Clustering
- Dimensionality reduction
- Forecasting
- NLP/text mining
- Recommendation systems
- Anomaly detection
- Association rules
- Causal vs predictive reasoning
- Decision science
- Model validation
- Leakage prevention
- Metrics and diagnostics
- Responsible AI
- Production readiness
- Model monitoring and drift
- Reporting and stakeholder communication

## Structure

```text
data-scientist/
├── SKILL.md
├── references/
├── templates/
├── checklists/
└── examples/
```

## Contents

- `SKILL.md`: skill entrypoint and routing guidance.
- `references/`: detailed operating guidance for methodology, model selection, validation, metrics, diagnostics, feature engineering, visualization, responsible AI, production readiness, monitoring, causal analysis, and decision science.
- `templates/`: reusable workflows and report/model/pipeline templates.
- `checklists/`: audit checklists for modeling, deployment, responsible AI, data quality, and method-specific review.
- `examples/`: practical mini-playbooks for Python, R, SQL, and Excel/Sheets.

## Purpose

The goal of this skill is to help an assistant behave like a senior data scientist: choosing the right method, validating models honestly, preventing leakage, interpreting results clearly, and connecting analysis to real business decisions.

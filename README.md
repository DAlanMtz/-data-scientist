# Data Scientist Skill
A senior data scientist OpenCode skill for data mining, model building, validation, interpretation, responsible AI, and production-readiness.
This skill is designed to support practical data science work across Python, R, SQL, Excel/Sheets, notebooks, scripts, databases, dashboards, APIs, and production pipelines.

## What It Covers
- Exploratory data analysis
- Data quality checks
- Feature engineering
- Model selection
- Regression
- Classification
- Clustering
- Dimensionality reduction
- Forecasting
- NLP/text mining
- Recommendation systems
- Anomaly detection
- Association rules
- Causal vs predictive reasoning
- Decision science
- Model validation
- Leakage prevention
- Metrics and diagnostics
- Responsible AI
- Production readiness
- Model monitoring and drift
- Reporting and stakeholder communication

## Purpose

The goal of this skill is to help an assistant behave like a senior data scientist: choosing the right method, validating models honestly, preventing leakage, interpreting results clearly, and connecting analysis to real business decisions.

## Tooling

The skill is intentionally tool-flexible. It can guide work in:

* Python
* R
* SQL
* Excel / Google Sheets
* Notebooks
* Scripts
* Databases
* Dashboards
* APIs
* Production pipelines

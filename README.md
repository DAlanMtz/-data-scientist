# Data Scientist OpenCode Skill

Senior, tool-flexible data science skill for data mining, exploratory analysis, model building, validation, interpretation, business decision-making, responsible AI, and production-readiness.

The skill supports work across Python, R, SQL, Excel/Sheets, notebooks, scripts, databases, dashboards, APIs, and production pipelines. It is not tied to a course, language, or single modeling tool.

## Contents

- `SKILL.md`: skill entrypoint and routing guidance.
- `references/`: detailed operating guidance for methodology, model selection, validation, metrics, diagnostics, feature engineering, visualization, responsible AI, production readiness, monitoring, causal analysis, and decision science.
- `templates/`: reusable workflows and report/model/pipeline templates.
- `checklists/`: audit checklists for modeling, deployment, responsible AI, data quality, and method-specific review.
- `examples/`: practical mini-playbooks for Python, R, SQL, and Excel/Sheets.
